The Internet vs The Web

The Internet and the World Wide Web are not the same thing

The Internet is the network infrastructure that enables the web to exist

The Web is a service that sits on top of the Internet

The Web

The protocol that makes the web possible is called Hypertext Transfer Protocol (HTTP)

HTTP is an application protocol

This simply means that it is a method by which applications talk to each other over a network
